# Employee REST API

A Flask-based REST API for managing employees/students.

## Features
- User Authentication
- CRUD Operations
- JSON Data Storage
- Docker Support
- Pytest Unit Testing
- GitHub CI Workflow

## Tech Stack
- Python
- Flask
- Pytest
- Docker
- GitHub Actions

## Run Locally

Install dependencies:
pip install -r requirements.txt

Run the app:
python run.py

App runs on:
http://127.0.0.1:5000

## Run Tests
pytest
